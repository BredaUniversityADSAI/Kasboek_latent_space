# Visual Design Composition idea  

# Overview  
Visual templates created for the Latent Space installation – an AI-driven art project producing personalized prints based on eye-tracking and emotional analysis.  

## Emotional state templates
Four template designs corresponding to emotional states:
- **Introspective** – Blue/purple gradient  
- **Peaceful** – Green/blue gradient  
- **Tempest** – Red/orange gradient  
- **Radiant** – Yellow/pink gradient

## Template Features
- Gradient visualization representing the emotional state  
- Designated area for eye-tracking overlay  
- Layout for AI-generated poem  
- Metadata section (depth %, duration, pattern ID)  
- Session ID and timestamp  

## Technical Specifications
- Format: A4 (210×297mm)
- Resolution: 300 DPI

## Purpose  
These templates serve as **design foundations and inspiration** for future iterations of the installation. They are ready for integration with eye-tracking data, poem insertion, and metadata overlay.
